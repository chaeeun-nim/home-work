## 마크다운 문법이란?

### 마크다운 정의

마크다운은 일반 텍스트 문서에 서식 요소를 추가하는 데 사용할 수 있는 가벼운 마크업 언어입니다. 2004년 존 그루버 가 만든 마크다운은 현재 세계에서 가장 인기 있는 마크업 언어 중 하나입니다.

### 마크다운을 사용하는 이유

- 다양한 문서 작성에 용이하다
- 모든 애플리케이션을 사용하여 열수 있을 정도로 이식성이 뛰어나다.
- 플랫폼이 독립적이라 모든 운영체제에서 작성 가능하다.
- Redit 과 GitHub 과 같이 많은 데스크톱/웹 기반 애플리케이션에서도 지원한다.

### 마크다운 문법 작동 원리

![마크다운 관련 설명이미지](../assets/images/markdown-flowchart.avif)

1.  텍스트 편집기나 전용 마크다운 애플리케이션을 활용하여 .md 또는 .markdown 확장자 파일을 생성한다
2.  마크다운 애플리케이션에서 마크다운 파일을 연다
3.  마크다운 애플리케이션이 마크다운 파일을 HTML문서로 변환한다.
4.  웹브라우저나 마크다운 애플리케이션을 사용하여 PDF와 같은 다은 파일 형식으로 변환한다.

## 마크다운 문법 사용 방법

### 헤더 : 문서의 제목 작성

- H1~H6 까지 되어있으며 H1은 문서의 제목으로 활용되고, 본문의 제목은 H2부터 활용하는 것이 좋다.
  헤더를 활용하여 문서를 구조화 하기때문에, 상위 섹션과 하위섹션이 무엇인지 잘 고려해서 사용하는 것이 좋다.
- **실전 사용방법**<br>

  ```bash
    # 제목내용 : H1 에 해당하며, #의 수를 늘려 H6까지 표현이 가능하다.
    ## 제목내용 : H2
    ### 제목내용 : H3
    #### 제목내용 : H4
    ##### 제목내용 : H5
    ###### 제목내용 : H6
  ```

- 예시

> ##### 이것은 H5에 해당합니다.

### 글자를 꾸며주는 기능.

#### 볼드체

- 특정 글자를 더 두껍게 표현하여 강조 효과를 줄 수 있다.
- **실전 사용방법**<br>

```bash
  **볼드체** : 단어의 양옆에 **를 붙여 작성한다.
```

- 예시

  > **이 글은 볼드체를 적용한 글입니다.**

---

#### 기울임

- 특정 글자를 기울임 꼴로 변형하여 표현한다. 볼드와 비슷하게 강조효과를 줄 수있다.
- **실전 사용방법**<br>

```bash
  *기울임* : 단어의 양옆에 *를 붙여 작성한다.
  _기울임_ : 단어의 양옆에 _를 붙여 작성한다.
```

- 예시<br>

  > _이 글은 기울임을 적용한 글입니다._

---

#### 취소선

- 특정단어를 취소하는 표기를 시각적으로 나타낼 수 있습니다.
- **실전 사용방법**<br>

```bash
  ~~취소선~~ : 단어의 양옆에 ~~를 붙여 작성한다.
```

- 예시<br>

> ~~이글은 취소선을 적용한 글입니다.~~

---

#### 하이라이트

- 특정 단어를 강조합니다. 형광펜으로 색칠한 듯한 시각적 효과를 줍니다.
- **실전 사용방법**<br>

```bash
  ==하이라이트== : 단어의 양옆에 ==를 붙여 작성한다.
```

- 예시<br>

  > 이글은 == 하이라이트 ==를 적용한 글입니다.

**특이하게도 이방법은 세간에 널리 알려져있지만, <mark>현 문서 작성자에게는 표현이 안되어 다른 표현방식</mark>을 알아보았다.**

- **실전 사용방법2**<br>

```bash
  <mark>하이라이트</mark> : mark태그를 활용해 작성한다.
```

- 예시<br>

  > <mark>이 글은 mark 태그를 활용한 하이라이트 입니다.</mark><br>

  > 이 방법은 마크다운 문법은 아니지만, 보통 웹브라우저를 활용하여
  > 마크다운으로 작성된 문서를 보기때문에 적용이 가능 한 게 아닌가 추측된다.

### 리스트

- 문서를 작성할때 리스트를 구조화하여 시각적으로 보기좋게 표현하기위해 사용한다. 현재 해당문서도 h3의 제목 아래에 설명 및 예시를 리스트로 표현하고있다.
- **실전 사용방법**<br>

```bash
  - 리스트 : -(대시라인)을 적고 탭을눌러 리스트를 제작한다.
    들여쓰기 + 대시라인은 리스트의 하위 항목도 작성 가능하다.
```

- 예시<br>

> - 리스트 1
>   - 리스트1의 하위항목
>     - 리스트 1의 하위항목의 하위항목
>
> 이런식으로 들여쓰기를 통해 하위항목을 표현할 수 있다.

### 인용문

- 문서에 인용문을 첨부할때 인용문임을 시각적으로 표현할때 사용한다.
  본래의 활용은 아니지만, 쉬운 이해를 위해 해당 문서의 예시 부분이 모두 인용문으로 작성되었다.
- **실전 사용방법**<br>

```bash
  > 인용문 : >를 적고 그뒤에 인용 내용을 적으면 인용문으로 작성가능하다.
```

- 예시<br>
  > 이글은 인용문으로 작성되었습니다.
  >
  > > 들여쓰기를 통해 중첩 인용문도 사용이 가능하다

### 구분선

- 문서의 단락과 단락을 구분해주는 역할을 한다. 현 문서는 기능에 따라서  
  시각적 분할을 위해 구분선을 사용하고 있다.
- **실전 사용방법**<br>

```bash
  --- : ---(대시라인 3개)를 작성하면 수평선이 그려진다.
  *** : ***를 작성하면 수평선이 그려진다.
```

- 예시<br>

  ***

  > 위의 수평선이 구분선 입니다.P

### 링크를 연결하는 방법

- 마크업 문서에 외부 링크를 연결할때 사용합니다.
- **실전 사용방법**<br>

```bash
  https://www.naver.com/ : 링크를 직접 입력하여도 링크이동이 가능하다.
  [텍스트](주소url) : 이와 같은 형식으로 입력하면, 원하는 텍스트에 링크이동도 가능하다.
```

- 예시<br>
  > [네이버](https://www.naver.com/)

### 코드박스

- 문서 작업내에 특정언어의 코드를 표현할때 사용한다.
- 인라인 코드와 코드박스 두가지 방법이 있다.
- **실전 사용방법**<br>

````bash
 `코드` : 코드 양옆에 `(백틱)을 작성한다. 인라인코드 표현 방법이다.
  ``` 언어 코드 ``` : ```(백틱3개) 로 영역을 설정하여 코드 박스를 만들 수 도 있다.
````

- 예시<br>

  > `<span> 인라인 코드 입니다. </span>`
  >
  > ```html
  > <span> 이것은 코드 박스입니다 </span>
  > ```

- 해당문서에 직접 작성되며 활용된 모습이다.
  ![코드박스 예시이미지](../assets/images/eximage.jpg)

- 코드박스에 다양한 언어를 삽입할 수 있는데, 다양한 언어를 삽입하기위한 마크다운 언어 표현 방식은 아래와 같다.<br>

|   언어    | 마크업 |    언어    |   마크업   | 언어 | 마크업 | 언어 | 마크업 |
| :-------: | :----: | :--------: | :--------: | :--: | :----: | :--: | :----: |
|   Bash    |  bash  |     C#     |     cs     | C++  |  cpp   | Diff |  diff  |
| HTML, XML |  html  |    HTTP    |    http    | Ini  |  ini   | JSON |  json  |
|   Java    |  java  | JavaScript | javascript | PHP  |  php   | Perl |  perl  |
|  Python   | python |    Ruby    |    ruby    | SQL  |  sql   |  --  |  -끝   |

### 테이블(표) 작성 방법

- 문서에 테이블(표)를 표현 할 수 있다.
- **실전 사용방법**<br>

```bash
|한셀|한셀|
|--|--|
|한셀|한셀|
위와 같은 양식으로 작성하면 표의 형식으로 표현할 수 있다.
대시라인 앞,뒤,양쪽에 : 를 붙여 정렬을 변경할 수 도 있다.
|한셀|한셀|
|:--|:--|
|한셀|한셀| -> 왼쪽정렬
ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ
|한셀|한셀|
|--:|--:|
|한셀|한셀| -> 오른쪽 정렬
ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ
|한셀|한셀|
|:--:|:--:|
|한셀|한셀| -> 가운데 정렬
```

- 예시<br>
  > | 제목    | 내용    | 설명    |
  > | ------- | ------- | ------- |
  > | 테스트1 | 테스트2 | 테스트3 |
  > | 테스트1 | 테스트2 | 테스트3 |
  > | 테스트1 | 테스트2 | 테스트3 |

### 마크다운 문법 무시

- 마크다운 문법에 사용된 기호를 텍스트 그대로 표현 가능하다.
- **실전 사용방법**<br>

```bash
\**볼드체** : 백슬래시를 마크다운 문법 앞에 사용하면 텍스트 그대로 표현 가능하다.
```

- 예시<br>

  > **볼드체** Asterisk 두개를 붙여 볼드체를 표현할 수 있는데 백스페이스를 붙이면 아래와같이 표현된다. <br>

  > \*\*볼드체\*\* 표현되지 않는 모습
